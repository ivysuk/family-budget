# 가족 공동통장 관리 웹앱

구글시트 + Apps Script(GAS)로 만든 무료 웹앱. 서버/호스팅 비용 없음.

## 구글시트
이미 만들어져 있음: https://docs.google.com/spreadsheets/d/19xNNA1L087hkZc39fDIJqnUsJBXBARytRCMkBOyosnU/edit

탭 구성: 가족구성원 / 월별납부 / 지출청구 / 렌탈료(정기지출) / 거래원장 / 잔액대사

**시작 전에 "가족구성원" 탭에 가족 이름을 먼저 입력해두세요.** (이 목록이 납부/청구 화면의 드롭다운에 나옵니다.)

## 배포 방법 (한 번만 하면 됨)

1. 위 시트 링크 열기
2. 상단 메뉴 **확장 프로그램 → Apps Script** 클릭
3. 왼쪽에 기본으로 있는 `Code.gs` 파일 내용을 전부 지우고, 이 저장소의 [`gas/Code.gs`](gas/Code.gs) 내용을 복사해서 붙여넣기
4. 왼쪽 파일 목록 옆 **+ → HTML** 클릭 → 파일명 `Index` 로 저장 (자동으로 `.html`이 붙음, `Index.html`이 되도록)
5. 방금 만든 `Index.html`에 [`gas/Index.html`](gas/Index.html) 내용을 복사해서 붙여넣기
6. 저장(디스크 아이콘 또는 Ctrl+S)
7. 우측 상단 **배포 → 새 배포**
8. 유형 선택(톱니바퀴) → **웹 앱**
9. "실행 계정": 나 / "액세스 권한이 있는 사용자": 가족들도 쓸 거면 "링크가 있는 모든 사용자" (또는 특정 구글 계정만 쓰게 하려면 그에 맞게 제한)
10. 배포 클릭 → 처음 한 번 권한 승인 필요 (본인 시트 접근이라 "고급 → 이동" 눌러도 안전)
11. 배포 완료 후 나오는 **웹 앱 URL**을 가족들에게 공유

## 데이터 흐름 요약

- **납부(수입)**: 가족이 매달 자기 형편에 맞는 금액을 직접 입력 → 자동으로 거래원장에 기록
- **지출 청구**: 가족이 먼저 돈을 쓰고 앱에서 청구 → 보스가 실제로 이체한 뒤 앱에서 "이체완료" 처리 → 자동으로 거래원장에 기록
- **거래원장**: 모든 입출금이 차변/대변 쌍으로 쌓이는 단일 기록 (복식부기)
- **잔액대사**: 거래원장 기준 계산된 잔액과 실제 통장 잔액을 입력해서 비교, 차이 확인

## 나중에 코드를 수정하고 싶으면

이 폴더(`gas/`)의 `.gs`/`.html` 파일을 수정한 뒤, Apps Script 편집기에 다시 붙여넣고 저장 → "배포 → 배포 관리 → 수정"으로 갱신하면 됩니다.
