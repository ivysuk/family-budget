# 가족 공동통장 관리 웹앱

**구조:** 화면(HTML)은 GitHub Pages에서, 데이터는 구글시트+Apps Script에서. 완전 무료.

```
사용자 브라우저 → docs/index.html (GitHub Pages, 진짜 웹사이트 URL)
                       ↓ fetch (데이터 조회/저장)
                   Apps Script 웹앱 (JSON API만 담당)
                       ↓
                   구글시트 (실제 데이터 저장소)
```

Apps Script가 직접 화면을 보여주던 이전 방식(못생긴 URL, 로딩 화면 튐)을 버리고, 화면과 데이터를 분리했습니다.

**시작 전에 "가족구성원" 탭에 가족 이름을 먼저 입력해두세요.**

## 1단계 — Apps Script 배포 (데이터 API)

1. 위 시트 링크 열기 → **확장 프로그램 → Apps Script**
2. `Code.gs` 내용을 전부 지우고 [`gas/Code.gs`](gas/Code.gs) 내용 붙여넣기
3. (이전에 `Index.html` 파일을 만들어두셨다면 왼쪽에서 삭제 — 이제 안 씀)
4. 저장 (Ctrl+S)
5. **배포 → 새 배포** → 유형: **웹 앱**
6. 실행 계정: 나 / 액세스 권한: **"링크가 있는 모든 사용자"** (이게 핵심 — 이렇게 해야 가족들이 구글 로그인 없이 그냥 URL로 바로 접속됩니다. "특정 계정만"으로 하면 로그인 오류가 납니다.)
7. 배포 → 권한 승인(본인 시트라 "고급 → 이동" 눌러도 안전) → 완료
8. 나오는 **웹 앱 URL** 복사 (`.../exec`로 끝남) — 2단계에서 씀

## 2단계 — 비밀번호 설정 (선택)

가족 외 사람이 URL을 알아도 못 들어오게 하고 싶으면:
1. 시트의 **"설정"** 탭 → "비밀번호" 행의 값 칸에 원하는 비밀번호 입력 (예: 1234)
2. 비워두면 비밀번호 없이 누구나 URL로 바로 접속 가능

## 3단계 — 화면(GitHub Pages) 배포

1. [`docs/index.html`](docs/index.html) 파일 맨 아래쪽 `const API_URL = 'PUT_YOUR_APPS_SCRIPT_WEB_APP_URL_HERE';` 부분을 1단계에서 복사한 웹 앱 URL로 교체
2. 이 프로젝트를 GitHub 저장소로 올리기 (아직 안 했다면 GitHub 계정 필요 — 무료)
   ```
   gh repo create family-budget --private --source=. --push
   ```
   (또는 GitHub 웹사이트에서 새 저장소 만들고 `git remote add origin <저장소주소>` 후 `git push -u origin master`)
3. GitHub 저장소 페이지 → **Settings → Pages**
4. Source: **Deploy from a branch**, Branch: **master**, 폴더: **/docs** 선택 → Save
5. 몇 분 뒤 `https://<계정명>.github.io/family-budget/` 주소로 접속 가능해짐 — 이게 가족들에게 공유할 진짜 URL

## 데이터 흐름 요약

- **초기 잔액**: 앱 쓰기 전 통장에 이미 있던 돈을 한 번 입력해두면 잔액 계산에 반영
- **월 목표**: 사람마다 매달 낼 목표 금액을 정해두면 게이지로 진행률 표시
- **납부(수입)**: 가족이 매달 형편에 맞게 직접 입력 → 자동으로 거래원장에 기록
- **지출 청구**: 가족이 먼저 쓰고 앱에서 청구 → 보스가 이체 후 "완료 처리" → 자동으로 거래원장에 기록
- **정기지출(렌탈료)**: 등록해두면 "청구" 탭에 매달 미입력 항목으로 뜸 → 그달 금액을 확인/수정 후 "청구하기"를 눌러야 실제 청구가 생성됨 (여러 명이 동시에 접속해도 자동 생성으로 중복되는 문제 방지, 매달 금액이 달라도 그때그때 입력 가능)
- **지출 항목**: 기본 4개(공과금/월세/생활비/기타) 외에 화면에서 직접 추가 가능
- **이달의 지출 통계**: 항목별 지출 비중을 막대로 표시
- **지난 기록 보기**: 상단 드롭다운으로 지난 달 데이터 조회 가능
- **거래원장**: 모든 입출금이 차변/대변 쌍으로 쌓이는 단일 기록 (복식부기)
- **잔액대사**: 시스템 계산 잔액과 실제 통장 잔액을 비교해서 차이 확인

## 코드 수정 후 반영하는 법

- `gas/Code.gs` 수정 → Apps Script 편집기에 다시 붙여넣기 → 저장 → **배포 → 배포 관리 → 수정(연필 아이콘) → 새 버전 → 배포**
- `docs/index.html` 수정 → git commit/push 하면 GitHub Pages가 자동으로 갱신 (1~2분 소요)
